## Description
> 1. <code>Spring Boot</code> + <code>MySQL</code> + <code>Data JPA</code> + <code>Spring Security</code>를 이용한 간단한 게시판 구현
> 2. 회원가입, 로그인, 게시글 작성, 수정, 삭제, 계층형 댓글 프로필 수정 기능
## 개발환경
|도구|버전|
|------|--------|
|Framework|Spring Boot 2.4.2|
|OS|Windows 10|
|IDE|IntelliJ IDEA Community|
|JDK|JDK 11|
